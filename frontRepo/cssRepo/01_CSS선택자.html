CSS 선택자

        선택자란?
        특정 HTML 요소를 선택할 때 사용

1. 모든 요소를 선택하는 선택자

        * : HTML 문서 내 모든 태그를 선택할 때 사용

        * {
            스타일 속성 : 값;
            스타일 속성 : 값;
            스타일 속성 : 값;
        }

2. 태그 선택자

        HTML 문서 내 같은 태그를 모두 선택
        태그명 {
            스타일 속성 : 값;
        }

3. 클래스 선택자

        HTML 문서 내 여러개의 태그를 동일한 클래스명으로 지정하여 선택
        .클래스명 {
            스타일 속성 : 값;
        }

4. 아이디 선택자

        HTML 문서 내 해당 아이디를 가진 태그를 선택
        #아이디 {
            스타일 속성 : 값;
        }
