배열

        JS는 타입이 없으므로, 타입 구분없이 데이터를 담을 수 있다.

1. 선인 및 초기화

2. 메소드


        indexOf()   :: 호출한 String 객체에서 주어진 값과 일치하는 첫 번째 인덱스를 반환 일치하는 값이 없으면 -1을 반환
        concat()    :: 호출 문자열에 문자열 인수를 이어 붙인 결과를 반환
        join()      :: 배열의 원소를 결합해서 하나의 문자열로 만든다
        reverse()   :: 배열의 순서를 뒤집어서 반전한다.
        sort()      :: 배열의 정렬 기본은 문자열기준 오름차순
        push()      :: 배열 끝에 여러 값을 추가
        pop()       :: 배열에서 마지막 요소를 제거하여 그 값을 호출자에게 반환
        shift()     :: 0번째 위치의 요소를 제거 하고 연이은 나머지 값들의 위치를 한칸 씩 앞으로 당깁니다. 그리고 제거된 값을 반환
        unshift()   :: 배열 형태의 객체 시작점에 주어진 값을 삽입
        slice()     :: begin부터 end 전까지의 복사본을 새로운 배열 객체로 반환한다. 즉, 원본 배열은 수정되지 않는다.
        splice()    :: 배열의 기존 요소를 삭제 또는 교체하거나 새 요소를 추가하여 배열의 내용을 변경한다. 이 메소드는 원본 배열 자체를 수정한다.
